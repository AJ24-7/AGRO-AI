# 🌿 AgroPilot AI – Smart Farming Assistant

AgroPilot AI helps farmers manage farms, analyze soil reports, get AI crop & 
fertilizer recommendations, detect crop diseases, and manage tractors — all 
from a modern green-themed dashboard.

> B.Tech Final Year Project | React + FastAPI + PostgreSQL + ML

---

## ✨ Features
- 🔐 JWT Authentication (Register / Login / Logout)
- 👤 Farmer Profile Management
- 🌾 Farm & Plot Management (CRUD)
- 🧪 Soil Report OCR (Tesseract) — extracts N, P, K, pH
- 🌱 AI Crop Recommendation (Scikit-Learn RandomForest)
- 🍃 Disease Detection + Treatment + History
- 🧴 Fertilizer Recommendation (Urea / DAP / Organic)
- 🚜 Tractor Management
- 🔔 In-App Notifications
- 📊 Dashboard Analytics with Recharts
- 🤖 AI Farming Chatbot
